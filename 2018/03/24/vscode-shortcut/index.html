Visual Studio Code快捷键

  • VsCode
Tool

Mark一下最常用的快捷键

标签页分组(最多3个)

  • Ctrl + \ 复制当前标签页到新组
  • Ctrl + 1 移动焦点到组1
  • Ctrl + 2 移动焦点到组2
  • Ctrl + 3 移动焦点到组3
  • Ctrl + Alt + -> 移动标签页到右侧组
  • Ctrl + Alt + <- 移动标签页到左侧组
  • Ctrl + PgUp 移动焦点到前一标签页
  • Ctrl + PgDn 移动焦点到后一标签页

工具

  • Ctrl + ` 内置命令行
  • Ctrl + Shift⇧ + P 输入命令

搜索

  • Ctrl + P 工作区文件搜索

显示

  • F11 全屏
  • Ctrl + + 放大
  • Ctrl + - 缩小
  • Ctrl + Num0 大小还原
