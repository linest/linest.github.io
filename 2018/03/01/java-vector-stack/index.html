Java Vector和Stack

  • 数据结构
  • 源码
Java

最原始的列表结构

Vector

Vector是抽象列表子列，从接口上看具备随机访问列表特性
内部除了底层数组，当前数据量，还有变量维护扩容增长量

容量如果不指定就是10，扩容增长是0即使用默认机制

增长的前提是保障容量，可能涉及扩容

所有操作都被同步保护

即使只读获取变量值也同步，因为变量值并无volatile修饰，保障可见性

Stack

Stack直接继承自Vector,实现上非常简单，利用最后一个元素作为栈顶



一分也是爱~
