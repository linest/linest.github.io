Visual Studio Code快捷键

  • VsCode
工具

弃用Atom转向Vs Code, Mark一下最常用的快捷键

标签页

  • Ctrl + Shift⇧ + N 新编辑器
  • Ctrl + W 关闭
  • Ctrl + ↑ / ↓ 滚动条
  • Ctrl + PgUp / PgDn 标签页切换

窗口

最多3组

  • Ctrl + \ 复制当前标签页到新组
  • Ctrl + 1 移动焦点到组1
  • Ctrl + 2 移动焦点到组2
  • Ctrl + 3 移动焦点到组3
  • Ctrl + Alt + → / ← 移动标签页到相邻组
  • Ctrl + PgUp / PgDn 移动焦点到相邻标签页

工具

  • Ctrl + ` 内置命令行
  • Ctrl + Shift⇧ + C 系统命令行
  • Ctrl + Shift⇧ + U 开关输出视图
  • Ctrl + Shift⇧ + P 输入命令
  • Ctrl + Shift⇧ + V 开Markdown预览

搜索

  • Ctrl + P 工作区文件名搜索
  • Ctrl + F 当前文件内搜索
  • Ctrl + H 当前文件内搜索替换

显示

  • F11 全屏
  • Ctrl + + 放大
  • Ctrl + - 缩小
  • Ctrl + Num0 大小还原

选中

  • Ctrl + F2 选中单词所有出现位置

行操作

  • Ctrl + Enter↵ 下方新行
  • Ctrl + Shift⇧ + Enter↵ 上方新行
  • Alt + ↑ / ↓ 移动行
  • Alt + Shift⇧ + ↑ / ↓ 赋值行

光标操作

  • Ctrl + G 跳到指定行
  • Home 跳到行首
  • End 跳到行尾
  • Ctrl + Home 跳到文件首
  • Ctrl + End 跳到文件尾
  • Ctrl + Shift⇧ + \ 跳到匹配括号

其他

  • Ctrl + / 注释


一分也是爱~
