Java TreeMap

  • 数据结构
  • 源码
Java

有序哈希表

有序依靠key比较机制

底层是一个比较器和树的根节点

SortedMap是有序接口
接口要求内含比较器，key有序存在最大最小，可根据key范围返回子视图，

NavigableMap是SortedMap的子接口，更加丰富了功能

比较器并不是必须的，可以无参构造
如果不提供比较器，那么key本身必须是可比较的，否则强转会报错

树是红黑树实现，树节点除了引用左右子树外，还引用父节点，默认是黑色
