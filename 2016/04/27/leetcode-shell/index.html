192. Word Frequency

词频统计，非常经典的问题

  • 首先cat先把要处理的内容显示出来，作为输入源
  • 接下来是分词。利用tr命令将空格处替换成换行，实现每个单词占一行。
    根据题意是空格分隔，但是考虑到有可能存在\r的情况，用[[:space:]]更严谨。
    复习一下各系统的换行符情况

\r = CR (Carriage Return) // Used as a new line character in Mac OS before X
\n = LF (Line Feed) // Used as a new line character in Unix/Mac OS X
\r\n = CR + LF // Used as a new line character in Windows

另外-s很重要，作用是压缩成一个，这样就可以处理连续空格这种情况,也能正确处理\r\n

  • 统计要利用uniq -c，但是它的前置条件是同类连续，所以先sort排个序
  • 最后sort -r处理成逆序，再awk调整一下单词和统计的显示顺序
