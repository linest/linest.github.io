Java TimeUnit

  • 源码
Java

1天=？秒

场景

传递时间的场景很常见，还经常需要传毫秒
虽然写成了乘积的形式，但是可读性还是欠佳

java.util.concurrent.TimeUnit提供时间粒度的便利转换

实现

TimeUnit是一个枚举类
每个时间单位是一个枚举实例，重写了一组不同时间单位间的转换方法
枚举类也可以定义成员和方法，TimeUnit是一种典型的应用

除了转换方法外，还提供了通用的包装方法wait,join,sleep
可以看出对底层方法进行了包装，调用了枚举类的转换函数转成毫秒

使用

  • 休眠3秒
  • 一天的秒数
