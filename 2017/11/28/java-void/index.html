Java Void

  • 关键字
  • 泛型
  • 源码
Java

void字面意思是空

作为关键字的void可以表示函数没有返回值

还有一个名为Void的类，不可实例化，作为关键字void的包装，内部只有一个类型

类型和关键字类型是一致的

在泛型里，如果泛型作为返回值，可以用Void表示没有返回值
因为不能实例化，唯一能接受的值就是null
