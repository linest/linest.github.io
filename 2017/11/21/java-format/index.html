Java Format体系

  • 源码
Java

信息的格式转换

基类java.text.Format

  • 日期(DateFormat)
  • 数字(NunmberFormat)
  • 消息(MessageFormat)

※ Format非线程安全，每个线程最好独立创建，可以搭配TheadLocal

Format最直观的用法是对象和字符串的相互转换
当然也可以提供位置信息等进行细粒度操作，由子类负责具体实现
内部采用StringBuffer实现
