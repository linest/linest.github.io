n个元素集合子集个数为2^n-1

❀77. Combinations

1~n数字中k个数字的组合

本身不重复，保持升序，每加入一个元素，后续必须更大

❀46. Permutations

给定一组不重复的数字，生成全部排列

想象成顺次挑选下一个数字
递归，每一层尝试所有前面没出现过的数字
复杂度O(n!)

迭代，每次对于前面所有可能，把新数字插入不同位置

❀47. Permutations II

给定一组可重复数字，生成全部排列

递归，由于有重复，无法简单判定是否前面已经用了，增加记录是否使用过
还需解决重复排列问题，先将数组排序，如果当前数字和前面相同，并且前面的未使用，那么当前数字也不用继续，因为前面数字使用时已经覆盖

迭代，依然是各位置插入，由于有重复，当插入位置和原来位置数字相同，那么停止后续，避免生成重复排列

❀949. Largest Time for Given Digits

给定4个数字，求能构成的最大时间，不能构成返回空串

求排列，因为个数只有四个，直接开循环，没必要递归
判定范围直接使用字符串比较，没必要转数字

401. Binary Watch

时间二进制，给定1的个数求所有时间组合

遍历所有时间组合，筛选1个数

❀78. Subsets

给定一组不重复数字，求全部组合

和排列不同

  • 不同顺序不会产生新结果，因此每层都向后尝试即可
  • 组合不定长，把过程也作为结果

迭代, 每次有两个选择，选这个数或者不选，即每一层都会在上一层基础上变为2倍数量

❀90. Subsets II

给定一组可重复数字，求全部组合

递归，先排序，同一层同一数字只处理一个即可，112组合已经包含12组合

迭代，对于重复元素，上轮没加的在下轮跳过，相当于只在增加的基础上增加
每层记录增加的部分

❀39. Combination Sum

一组不重复正数，同一数字可以重复利用，求和等于指定值的组合

可以重复使用，因此每层不会直接跳过这个数

迭代，同一值尽管加，直到超过目标

❀40. Combination Sum II

一组可重复数字，数字不可重复利用，求和等于指定值的组合

先排序，让重复数字排在一起，遇到重复数字，只有第一次进行尝试，避免重复组合出现

❀216. Combination Sum III

给1~9， 求k个数和为n的组合

数字不重复，也不能重用，生成排列，同时考虑数字个数和目标和

❀131. Palindrome Partitioning

分割一个字符串，使得每部分都是回文，求所有分割方法

字符串分割，实际上是个集合问题，插板法，从全部索引中选出若干个作为分割点，判断每部分是否是回文

❀22. Generate Parentheses

n对括号，求所有合法序列

合法序列任意前缀左括号大于等于右括号

DP卡特兰数也可以求解

❀784. Letter Case Permutation

数字字母混合字符串，输出所有字母大小写可能

BFS迭代，初始把原始串进队列，针对字母每一位，把之前结果处理为大写小写两个版本进队列

DFS递归，不断进行大小写分支

❀453. Minimum Moves to Equal Array Elements

一个数组n个数字，每次让n-1个数字都自增，求让数组相等的最小次数

暴力解法，每次选一个位置不变，其他增加
实际策略，经过m次操作全部达到x，那么sum+m*(n-1) = n*x，并且最小元素每次都会参与增加，即min+m = x
解出m = sum - n*min

293. Flip Game

只包含±的字符串，操作一次把两个+变成两个-，求所有可能

回溯，遇见两个+就变，复制加入结果后复原

❀294. Flip Game II

使对手无法操作就会胜利，求先手能否必胜

遍历每一种情况，递归判定

计算过程中会重复计算，可以建立哈希表缓存

❀15. 3Sum

一组可重复数字，找出所有3个数和为0的不重复组合

数字可重复，为了避免重复组合，排序是必须的
以每个数字为起点，向后找两个数字和，转化为2sum问题
因为已经有序，2sum可采用双指针法，注意一旦找到，还需要跳过后续重复
总时间复杂度O(n2)

93. Restore IP Addresses

限制条件：四部分，255以内，不能0开头

17. Letter Combinations of a Phone Number
