总结

关注点

  • 范围

技巧

  • 范围有限可以考虑使用数组

  • map.getOrDefault和map.putIfAbsent简化判断
  • 针对两个对象，无需维护两份统计，可以增减计数

判断空格分割出的单词是否符合模式，比如abba模式cc dd dd cc

806. Number of Lines To Write String

每个字母长度不等，每行长度最多100，求字符串占几行以及最后一行多长
注意写不下换行，每行不一定占满

804. Unique Morse Code Words

884. Uncommon Words from Two Sentences

383. Ransom Note

500. Keyboard Row
