注意条件一致不能混用

  • 二分搜索树，三元运算符
  • 全局辅助头结点，避免空头判断
  • 辅助方法递归，传递额外参数，基本类型或者引用本身还是要使用成员作为全局变量
    • 全局变量修改，那么无论是什么层级的递归都可见
    • 而基本传参，各层级都有副本，无法更改
    • 全局变量注意使用前先初始化
    • 基本类型也可以使用数组，自定义对象等来规避全局变量，将传递参数变成恒定引用的内容
  • 递归遍历
    • 通过递归层级处理深度
  • 迭代遍历
    • 通过同层计数处理深度

注意
错误的构建方式，引用值传递，传入方法的参数是一个副本，方法内的对象并不会接在外面

node = new Node();
