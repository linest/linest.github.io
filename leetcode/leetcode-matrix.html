LeetCode--矩阵篇

总结

关注点

  • 矩阵长宽，空数组
  • 一维二维映射 (idx / col, idx % col)
  • 角作为判定点

事项

二维数组，看做一维数组做容器，每个数据都是一个数组

下标前行后列

矩阵空判定

有序矩阵

❀74. Search a 2D Matrix

每行内有序，每行开头都比上一行末尾大，即行间有序，搜索一个数

下标映射二维看做一维，二分搜索

240. Search a 2D Matrix II

每行递增，每列递增，即右下方大于左上方

以右上角作为参照，收缩坐标范围，普通点变大可以向右也可以向下，右上角限制了只能向下
如果右上角大于目标值，右上角下面列更大被排除；如果右上角小于目标值，右上角左边行更小被排除；整体上矩阵范围不断缩小

螺旋遍历

❀54. Spiral Matrix

螺旋打印矩阵

时刻维护四个边界，按照右下左上顺序依次打印，每次走到允许的最大范围

单行单列情况下避免重复打需要判断
测试用例：空，单元素，单行，单列，多行多列

59. Spiral Matrix II

生成n*n的螺旋数组

螺旋遍历的同时自增填值

矩阵尺寸变动

867. Transpose Matrix

矩阵转置，行变列

长宽可能不一样，不能原地做，需要新开数组

566. Reshape the Matrix

改变矩阵长宽，如果不能则放弃操作

计算原数据映射到新矩阵中的位置

矩阵原地操作

832. Flipping an Image

01矩阵，每行前后翻转，并且数据01互换

按行操作，先翻转再使用异或互换

❀73. Set Matrix Zeroes

将含有0的行列都置为0

O(mn)空间，另开一个新数组初始化非0，原数组每遇到0，将新数组行列置0
O(m+n)空间，另开两个数组，作为行和列的标记
O(1)空间，利用原数组首行首列做标记，另有两个变量记录首行首列是否置0，先处理内部置0，最后决定边框

❀48. Rotate Image

顺时针旋转90度，原地操作

环状转移，建立每个位置旋转90度后与目标位置关系，做一次环形转移
矩阵看成若干个环，每个环的上面行进行一次环转移即可

先矩阵转置，再左右交换即可

矩阵邻接操作

❀661. Image Smoother

图像平滑，每个元素是环绕元素的均值

值相互影响，不能原地操作，需要新开矩阵
每个位点为中心强行3*3环绕遍历，忽略越界点

矩阵内容判定

❀766. Toeplitz Matrix

判断是否斜线上数据都相等

不需要整个斜线判断，只需要单个数据和左上角比较即可

如果内存只能加载一行，那么以行为基准，流式加载后续进行比较并且更新基准
如果内存载入一行的一部分，那么矩阵纵向分割，分割边界重叠保证边界也进行了判定

36. Valid Sudoku

部分填充数字的数独，验证是否合法

行，列，3*3块遍历
