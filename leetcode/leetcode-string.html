LeetCode-字符串篇

总结

可视为特殊类型数组，toCharArray()转换

关注

  • null
  • 空串
  • 大小写
  • 长度

操作

  • 正则

技巧

  • ASCII码a 97 A 65
  • char可以直接赋值数字常量，但是赋值int需要显式转换
  • Character提供工具方法isLetter,isDigit,isLetterOrDigit,toUpperCase,toLowerCase
  • 数字转字符转字符串 “”+ ‘a’ + i
  • String可以截取部分数组进行构造String(char value[], int offset, int count)
  • String可以指定起始位置搜索，indexOf(String str, int fromIndex)
  • String空白字符分割split("\W+")
  • StringBuilder append(String str)/insert(int offset, String str)

注意

  • String取长度是方法length(), 不是数组直接length

匹配

925. Long Pressed Name

输入时可能同一字符打了多次，即判断两个字符串是否是字母顺序相同，个数不同的串

当前输入字符要么匹配目标当前或者之前

❀844. Backspace String Compare

井号表示退格，判断两个字符串是否相同

直观方法是构建真实字符串，消耗空间
可以从末尾开始比较，累积退格次数，抵消字符

❀859. Buddy Strings

判断是否可以交换A串的两个字符后等于B串

  • 长度不等，不可能
  • 串相等，如果A存在相同字符
  • 串不等，A和B只有两个位置不同，并且交换后相等

❀49. Group Anagrams

单词分组

将每个单词排序规范化，作为标识

大小写

520. Detect Capital

判定大小写是否规范，要么全小写全大写，要么大写首字母

只统计小写个数即可，保证字母条件，单边判断即可

循环移位

❀796. Rotate String

判定两个字符串是否循环移位异构体

首先要保证两个串相等，倍增后所有单倍长度子串都是循环移位异构

单词

58. Length of Last Word

只包含字母和空格，求最后单词的长度

从末尾遍历，先排除末尾空格，然后开始统计字母

❀434. Number of Segments in a String

统计单词数

以结尾分界处作为判断基准，当前非空格并且后面是空格

❀819. Most Common Word

求段落中非禁用词中词频最高的单词

使用split分词加统计

回文

❀5. Longest Palindromic Substring

最长回文子串

尝试以每个位置作为中心，向两侧扩展，复杂度O(n2)

❀125. Valid Palindrome

忽略大小写和其他字符判断字母回文

先整体转小写，再双指针跳过无关字符判断

❀680. Valid Palindrome II

最多允许删除一个字符，判断是否能构成回文

发现两侧不相等，可以跳过左侧也可以跳过右侧，继续判断回文

246. Strobogrammatic Number

数字构成字符串，判断旋转180度后是否还是自身

能旋转的数字只有5个，建立哈希表，回文的一种变形

格式化

482. License Key Formatting

字符串重新分成长度k的组，用横线连接，第一组可以短

从末尾开始构造，注意横线出现条件

前缀

❀14. Longest Common Prefix

多个字符串最长公共前缀

选第一个作为基准，其他串依次比较

❀720. Longest Word in Dictionary

能逐个字母生成的最长单词，即所有前缀都存在的最长单词,多个结果时取字母序小的

排序，每次只需比较除去末尾的字符串是否存在

建立前缀树Trie，然后BFS遍历， O(n)

搜索

❀758. Bold Words in String

给一个单词列表，和一个字符串，<b></b>加粗字符串内所有单词，单词可以重复匹配，加粗可以合并

单独开一个数组记录这个位置是否要加粗
对于每个单词，indexOf找出所有出现，并且对单词所有位置设置标记
根据标记，连续部分插入加粗符号

翻转

344. Reverse String

翻转字符串

单指针版，也可以双指针

使用库

541. Reverse String II

每2k个长度为一组，翻转前k个，不足k个全翻转

❀917. Reverse Only Letters

翻转字符串，非字母不动

双指针法，跳跃非字母时要保持i,j约束

345. Reverse Vowels of a String

交换元音字母

双指针

❀151. Reverse Words in a String

以单词为单位翻转

注意收尾可能有空格，单词间可能多个空格
整体思路是先去除收尾空格和单词间多余空格
整体反转后每个单词再反转

应用

❀71. Simplify Path

简化路径表达式

因为存在..，可能抹掉前面，因此采用栈辅助

字母序比较

❀555. Split Concatenated Strings

给一组字符串，每个字符串可以选择翻转，拼成一个环形字符串，现从某一点切开，求最大字符串

某一个字符串内包含切点，只有包含切点字符串不确定，其余字符串应该以大字母序存在

当前方案是每个点都尝试，可以优化一下，如果开头比当前结果还小，直接跳过
